## :zap: Installation

1. [Clone / Download](https://github.com/Fyntune/eb-frontend.git) repo.
2. Inside the project open a terminal and run:
    ```
    npm install
    ```
    This will install all the project dependencies.
3. To start the development server run:
    ```
    npm start
    ```
